# Signal Calculator

A clean, modern glass-style calculator web app with:

- persistent history for previous calculations
- saved numeric variables for more complex formulas
- keyboard support for fast entry
- support for functions like `sin`, `cos`, `tan`, `sqrt`, `log`, `ln`, `pow`, `min`, and `max`

## Run it

Open `index.html` in a web browser.

## Tips

- Press `Enter` in the main expression box to calculate.
- Press `Ctrl+H` to open or close history.
- Save a variable directly with an assignment such as `rate = 12.5`.
- Use `ans` to reference the previous result.
- Use `pi` or `e` as constants.
